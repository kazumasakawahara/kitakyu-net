# WAM NET利用ガイド

## WAM NETとは

WAM NET（ワムネット）は、独立行政法人福祉医療機構が運営する、福祉・保健・医療の総合情報サイトです。全国の障害福祉サービス事業所の情報が公開されており、事業所検索や情報開示が行えます。

- 公式サイト: https://www.wam.go.jp/

## 事業所検索の利用方法

### アクセス手順

1. **WAM NET公式サイトにアクセス**
   - URL: https://www.wam.go.jp/

2. **障害福祉サービス等情報検索へ移動**
   - トップページから「障害福祉」→「事業所を探す」を選択
   - または直接URL: https://www.wam.go.jp/sfkohyoout/COP000100E0000.do

3. **検索条件の設定**
   ```
   都道府県: 福岡県
   市区町村: 北九州市
   サービス種別: （必要に応じて選択）
   ```

4. **検索実行**
   - 「検索」ボタンをクリック

### 検索結果の確認

検索結果画面では以下の情報が表示されます：
- 事業所名
- 法人名
- 所在地
- 電話番号
- サービス種別
- 定員
- その他の基本情報

### データのダウンロード

#### CSVダウンロード手順

1. 検索結果画面の上部にある「CSV出力」ボタンをクリック
2. ファイルがダウンロードされる
3. ファイル名の例: `sfkohyo_YYYYMMDD.csv`

#### ダウンロードファイルの保存先

```
~/AI-Workspace/kitakyu-net/data/raw/
```

#### ファイル命名規則

```
wamnet_kitakyushu_{service_type}_{YYYYMMDD}.csv

例:
- wamnet_kitakyushu_all_20251031.csv
- wamnet_kitakyushu_就労継続B型_20251031.csv
```

## データ項目の対応表

### WAM NETの項目名とシステムの項目名の対応

| WAM NET項目名 | システム項目名(英語) | システム項目名(日本語) | 備考 |
|--------------|-------------------|-------------------|------|
| 事業所番号 | facility_number | 事業所番号 | 10桁の数字 |
| 事業所名 | name | 事業所名 | - |
| 法人名 | corporation_name | 法人名 | - |
| サービス種類 | service_type | サービス種別 | - |
| 郵便番号 | postal_code | 郵便番号 | XXX-XXXX形式 |
| 所在地 | address | 住所 | 完全な住所 |
| 電話番号 | phone | 電話番号 | - |
| FAX番号 | fax | FAX番号 | - |
| 定員 | capacity | 定員 | 数値 |

### 追加で収集が必要な項目

WAM NETには含まれないが、システムで管理する項目：
- 所在区（住所から抽出）
- 空き状況（手動入力）
- 営業日・営業時間（個別確認）
- 対応障害種別（個別確認）
- 送迎の有無（個別確認）
- その他の詳細情報

## サービス種別ごとの検索

### 全サービス種別の一括検索

```
検索条件:
- 都道府県: 福岡県
- 市区町村: 北九州市
- サービス種別: （選択しない）
```

これにより、北九州市の全事業所を一度に取得できます。

### 個別サービス種別での検索

特定のサービス種別のみが必要な場合：

#### 例1: 就労継続支援B型のみ
```
サービス種別: 就労継続支援B型
```

#### 例2: 生活介護のみ
```
サービス種別: 生活介護
```

## データの鮮度と更新頻度

### WAM NETの更新タイミング

- 事業所情報は随時更新される
- 新規指定事業所は指定後に追加
- 廃止事業所は廃止後に削除

### 推奨される取得頻度

- **定期取得**: 月1回
- **新規事業所確認**: 四半期に1回
- **緊急時**: 必要に応じて随時

## データ取得の注意事項

### 1. データの信頼性

- WAM NETのデータは公式情報に基づく
- ただし、リアルタイムではない可能性
- 重要な情報は事業所に直接確認を推奨

### 2. 個人情報の取り扱い

- WAM NETのデータは公開情報
- ただし、収集後の管理には注意
- 個人を特定できる情報の追記は慎重に

### 3. 著作権・利用規約

- WAM NETの利用規約を遵守
- データの二次利用には注意
- 営利目的での利用は制限される場合あり

## データ整形の必要性

### WAM NETからダウンロードしたCSVの特徴

1. **文字コード**: Shift-JIS（CP932）
2. **改行コード**: CR+LF（Windows形式）
3. **列区切り**: カンマ
4. **ヘッダー**: 日本語

### 整形が必要な理由

- 文字コードをUTF-8に変換
- 列名を英語名に変換
- データ型の適切な変換
- 欠損値の処理
- 重複データの除去

## トラブルシューティング

### 問題1: 検索結果が表示されない

**原因:**
- 検索条件が厳しすぎる
- システムのメンテナンス中

**対処:**
- 検索条件を緩和する
- 時間をおいて再試行

### 問題2: CSVダウンロードができない

**原因:**
- ブラウザの設定
- ポップアップブロック

**対処:**
- ポップアップを許可
- 別のブラウザで試す

### 問題3: ダウンロードしたCSVが文字化け

**原因:**
- 文字コードの不一致

**対処:**
- Shift-JISで開く
- データ処理スクリプトで自動変換

## 自動化の可能性

### 現状の制約

WAM NETのサイトは：
- 公式APIは提供されていない
- Webスクレイピングには注意が必要
- 利用規約の確認が必須

### 推奨される方法

1. **手動ダウンロード + 自動処理**
   - WAM NETからCSVを手動ダウンロード
   - ダウンロード後の処理は自動化

2. **定期的な手動確認**
   - 月1回程度の手動ダウンロード
   - データの差分チェックは自動化
